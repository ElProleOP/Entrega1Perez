Este proyecto es una pagina web de una Libreria donde el usuario puede registrar libros, juegos de mesa,
hacerse socio o buscarse como socio a traves de un email asociado

1/ La clase Libros consta de 3 caracteristicas: genero, titulo y un numero que sirve de guia. 
Para acceder a la seccion de registrar un libro se clickea en la barra de arriba (Libros) 

2/ La clase Juegomesa consta de 3 caracteristicas: titulo, precio y un numero que sirve de guia.
Para acceder a la seccion de registrar un juego de mesa se clickea en la barra de arriba (Juego de mesa) 

3/ La clase Socio consta de 4 caracteristicas: nombre, apellido, edad y un email. 
Para acceder a la seccion de registrarse como socio se clickea en la barra de arriba (Socio)

4/ Por ultimo esta la seccion Buscar Socio, que sirve para buscar los datos de un socio en la base de datos
asociado al email.
Para acceder a la seccion de Buscar socio se clickea en la barra de arriba(Buscar Socio)
